# 내 목소리 프리셋 만들기 (유튜브/강의용)

본인 목소리를 클론해서 텍스트만 입력하면 내 목소리로 음성을 생성하는 가이드입니다.

---

## 1단계: 음성 녹음 (가장 중요)

### 녹음 환경
- **조용한 방** — 에어컨, 선풍기 끄기
- **마이크** — 헤드셋/에어팟도 OK, 콘덴서 마이크면 최상
- **거리** — 마이크에서 15~20cm

### 녹음 방법 (2가지)

**방법 A: 웹 UI에서 직접 녹음 (가장 간편)**
1. `http://localhost:3077` 접속
2. "새 음성 추가" → 마이크 아이콘 클릭
3. **20~30초** 동안 아래 스크립트 읽기
4. 녹음 완료 → 자동 업로드

**방법 B: 별도 녹음 후 업로드**
1. 녹음 앱 (Mac: QuickTime, 핸드폰: 음성메모)으로 녹음
2. WAV 또는 M4A로 저장
3. 웹 UI에서 파일 업로드

### 녹음 스크립트 (20초)

아래 텍스트를 **자연스럽게, 평소 말투로** 읽으세요:

> 안녕하세요, 김지원입니다. 오늘은 프로그래밍에 대해 이야기해볼까 합니다.
> 개발을 하다 보면 다양한 문제를 만나게 되는데요,
> 이런 문제들을 하나씩 해결해나가는 과정이 정말 재미있습니다.
> 그럼 지금부터 시작해보겠습니다.

### 녹음 팁
- **평소 말투** 그대로 — 아나운서처럼 할 필요 없음
- **일정한 속도** — 너무 빠르거나 느리지 않게
- **끊지 않고** 한 번에 쭉 읽기
- 기침, 음... 같은 소리 피하기
- 여러 번 녹음해서 가장 자연스러운 것 선택

---

## 2단계: 음성 준비 & 생성 테스트

1. 음성 업로드/녹음 완료 후 → "음성 준비" 자동 진행
2. 텍스트 입력란에 테스트 문장 입력:
   ```
   안녕하세요, 이 영상에서는 리액트와 타입스크립트를 활용한 프론트엔드 개발에 대해 알아보겠습니다.
   ```
3. "음성 생성" 클릭 → 결과 확인
4. **만족스럽지 않다면**:
   - `exaggeration` 값 조정 (0.3~0.7 사이 실험)
   - `temperature` 낮추기 (0.6~0.8) → 더 안정적인 음성
   - 다른 녹음 파일로 재시도

---

## 3단계: 프리셋으로 저장

결과가 만족스러우면:

1. **"프리셋 저장"** 버튼 클릭
2. 이름: `내 목소리 (강의용)` 또는 원하는 이름
3. 성별/연령/톤 선택
4. 저장 완료 → 다음부터 프리셋 탭에서 바로 로드

---

## 4단계: 유튜브/강의에 활용

### 활용 시나리오

**A. 유튜브 나레이션**
1. 프리셋 로드
2. 대본 텍스트 입력 (문장 단위로 자동 분리됨)
3. 음성 생성 → 다운로드
4. 영상 편집 프로그램에서 삽입

**B. 여러 톤의 내 목소리 만들기**
같은 프리셋에서 파라미터만 바꿔서 저장:
- `내 목소리 (차분한)` — speed: 0.9, pitch: -2
- `내 목소리 (활기찬)` — speed: 1.1, pitch: +1
- `내 목소리 (강의용)` — speed: 1.0, pitch: 0

**C. 긴 대본 배치 처리**
1. 텍스트 큐에 여러 문장 추가
2. "전체 생성" → 자동 순차 처리
3. 각각 다운로드

### 파라미터 추천 (강의용)

| 파라미터 | 값 | 이유 |
|----------|-----|------|
| exaggeration | 0.4~0.5 | 너무 극적이지 않은 자연스러운 톤 |
| cfg_weight | 0.5 | 기본값 유지 |
| temperature | 0.7 | 안정적이면서 단조롭지 않은 |
| speed | 0.95~1.0 | 강의는 약간 천천히가 좋음 |
| pitch_semitones | 0 | 본인 목소리 그대로 |

## 5단계: 더 높은 품질을 원한다면

### Fish Speech S2로 클론 (CUDA GPU 필요)
Fish Speech는 더 많은 언어(47개)와 빠른 속도를 지원합니다:
1. 엔진 드롭다운에서 "Fish Audio S2" 선택
2. 같은 음성 파일 업로드 + 전사 텍스트 입력
3. 프리셋 저장 → Chatterbox 버전과 비교

### 외부 서비스 보조 활용
- **Fish Audio** ($5.50/월) — 이미 프로젝트에 통합됨, API로도 사용 가능
- **ElevenLabs** ($22/월 Creator) — Professional Voice Clone으로 거의 완벽 복제
- **Supertone Play** — 한국어 자연스러움 최상

## 트러블슈팅

### "음성이 로봇 같아요"
- 녹음 품질 문제가 가장 흔한 원인
- 더 조용한 환경에서 재녹음
- exaggeration을 0.5~0.7로 올려보기

### "내 목소리와 다르게 들려요"
- 녹음 시간을 30초 이상으로 늘려보기
- temperature를 0.6으로 낮춰보기
- 여러 번 생성해서 비교 (매번 약간씩 다름)

### "특정 단어 발음이 이상해요"
- 해당 단어를 발음 나는 대로 적기 (예: "Next.js" → "넥스트 제이에스")
- 문장을 짧게 나누기
